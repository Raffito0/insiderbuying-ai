## CRITICAL RULES

1. **Data first, ideas second.** Do ALL searches in Phase 1-5 before proposing any idea. Ideas must EMERGE from evidence, not be invented and then confirmed.

2. **EVERY revenue claim needs a URL.** "NO EVIDENCE FOUND" is honest and acceptable. Fabrication is not.

3. **The brand must NOT be boring.** If the content wouldn't make a 25-year-old stop scrolling, it's wrong. Modern, sharp, personality-driven. Data-backed hot takes > generic tips. Think Pieter Levels energy, not LinkedIn corporate.

4. **Monetize from week 1-2.** Brand building is allowed IF there's a monetization mechanism from the start (product in bio, affiliate links, paid Telegram, Gumroad product). "Grow first, monetize later" is BANNED.

5. **The agent does EVERYTHING.** If an idea requires me to write tweets, respond to DMs, create content, or handle support — it fails P1. The agent must handle all of it autonomously.

6. **Be honest about X growth speed.** Don't assume 10K followers in month 1. Research what's realistic for automated accounts in each niche. Use real data.

7. **Viral surfing is a hypothesis — validate it.** Can a Python script detect early viral tweets? How many impressions does a first reply get? Research this before building ideas around it.

8. **Mix of ideas welcome.** The output can include brand-building plays, direct-reply-selling plays, SaaS plays, data product plays, bot plays — whatever the data supports. Don't force everything into one model.

9. **Multiple products per brand is fine.** A single X brand can drive traffic to Apify + Gumroad + Telegram bot simultaneously. Explore this if the data supports it.

10. **Conversion math must be conservative.** Bio link CTR: use 1-2% (not 5%). Reply-to-click: use 2-3% (not 10%). Click-to-buy: use 1-3% (not 10%). Show pessimistic AND optimistic scenarios.
